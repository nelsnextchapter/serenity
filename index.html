Developer Notes — Adding Content

Add a new meditation: Open serenity.html in a text editor. Find the MEDITATIONS array near the top of the <script> tag. Add a new object with: name, emoji, cat (anxiety/sleep/morning/healing/focus/love), color (CSS gradient), dur (minutes), desc.

Add a new sound tile: Find the SOUNDS array. Add: name, emoji, id (unique lowercase slug). Then upload an audio file above and assign it to your new slot.

Assign audio to a built-in meditation: Upload the file above, then choose the meditation name from the "Meditation slot" dropdown and click Save.
